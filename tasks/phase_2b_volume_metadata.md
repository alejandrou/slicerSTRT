# Phase 2B - Volume Metadata Inspection

## Goal

Add a small feature that inspects the currently selected volume and displays basic metadata.

## Feature Requirements

- Selected volume name.
- Image dimensions.
- Spacing.
- Origin.
- Direction/orientation if safely available.
- Scalar type if safely available.
- Voxel count or estimated size if simple to compute.
- Clear message when no volume is selected.

## Non-Goals

- No DICOM import.
- No segmentation.
- No AI.
- No report generation.
- No heavy refactor.
