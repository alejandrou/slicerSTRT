# Phase 2C - Template Cleanup

## Goal

Remove or rename remaining generated thresholding-template code, labels, comments, and tests that no longer match STRATUM.

## Feature Requirements

- Remove misleading threshold-volume wording.
- Rename UI text to STRATUM/environment/inspection concepts.
- Keep the module loadable.
- Keep changes small and reviewable.
- Do not introduce new STRATUM features during cleanup.

## Non-Goals

- No new functionality.
- No UI redesign.
- No large architecture changes.
