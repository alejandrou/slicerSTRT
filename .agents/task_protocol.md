# Task Protocol

Use this protocol for normal Codex work in this repository.

- Make small, reviewable changes.
- Prefer one feature slice per task.
- Do not refactor unrelated code.
- Do not make large rewrites unless explicitly approved.
- Do not run Git commands unless explicitly requested.
- Do not edit generated or build folders.
- Explain changes simply.

After each task, report:

1. Files changed.
2. What changed.
3. Why it changed.
4. How to test manually.
5. Remaining risks or the next recommended step.
