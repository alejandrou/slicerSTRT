# Knowledge

This folder is for curated, non-sensitive, version-controlled reference notes useful to the STRATUM-related Slicer prototype.

Use `docs/knowledge/` for concise project reference notes that are safe and useful for the repository history.

Use root `knowledge/` for ignored local reference material, downloaded files, and private working notes. `knowledge/` is protected local material and is not version controlled.

Do not store private medical data, private hospital data, secrets, personal notes, or bulky downloaded references in `docs/knowledge/`. Do not treat either knowledge folder as an implementation area or as an alternate owner for repository policies and workflows.
